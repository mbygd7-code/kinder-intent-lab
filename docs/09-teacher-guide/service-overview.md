# Kinder Intent Brain — 서비스 설명서 (선생님용)

> 이 문서는 **"이게 뭐예요?"** 에 답합니다.
> 화면 사용법은 『사용설명서』(user-manual.md)를 봐주세요.

## 🌟 왜 이 뇌를 키워야 하나요? — 먼저 읽어주세요

**한 줄: 선생님이 가르친 만큼, 킨더버스가 "말귀 알아듣는 서비스"로 바뀝니다.**
하루 수십 번의 메뉴 찾기·반복 클릭이 평소 말투 한마디로 줄어드는 것 — 그 미래는
선생님의 답 없이는 오지 않습니다.

### 장면으로 보면 — 하원 30분 전

오늘 물감놀이 사진 30장을 학부모께 보내야 합니다. 잘 나온 컷을 고르고, 다른 반 아이
얼굴을 가리고, 알림장을 쓰고, 발송까지.

| | 어떻게 하나 |
|---|---|
| **지금** | 사진첩 → 잘 나온 것 고르기 → 편집 → 얼굴 가리기 → 알림장 쓰기 → 첨부 → 대상 선택 → 발송… **메뉴 4곳, 수십 번 클릭** |
| **뇌가 자라면** | 🗣 **"오늘 물감놀이 사진 잘 나온 것만 얼굴 가려서 부모님들께 보내줘"** — 뇌가 순서대로 준비하고, 선생님은 **마지막 확인만** |

같은 방식으로 이런 말들이 전부 "한마디 업무"가 됩니다:

| 이렇게 말하면 | 뇌가 알아듣는 업무 |
|---|---|
| 🗣 "지우 오늘 병결로 해줘" | 출결 기록 |
| 🗣 "작년 운동회 계획안 파일 어디 있지?" | 자료 검색 |
| 🗣 "이 알림장 내일 아침에 나가게 해줘" | 예약 발송 |
| 🗣 "방금 그 상황 기록해줘" | 관찰 기록 |
| 🗣 "뒤에 다른 반 애들 얼굴 좀 가려줘" | 얼굴 모자이크 |
| 🗣 "내일 견학 가니까 준비물 알림장 좀 써줘" | 알림장 초안 |

화면의 **점 63개가 전부 이런 업무 하나씩**입니다 — 뇌가 배우는 과목 63개.

### 선생님께 돌아오는 것

- **⏱ 시간** — 위 장면처럼 여러 단계 업무가 한마디로. 줄어든 잔업 시간은 아이들 곁으로
  돌아갑니다.
- **🙋 내게 맞춤** — 같은 "알림장 정리해줘"도 A선생님께는 **문장 다듬기**, B선생님께는
  **사진 배치**일 수 있습니다. 쓸수록 **내 뜻**으로 알아듣습니다. (성향이 뇌 판단에 주는
  영향엔 상한선이 있어 특정 말버릇에 치우치지 않아요.)
- **🛡 실수 방지** — 학부모 전송·출결 기재·자료 삭제처럼 **되돌릴 수 없는 7가지**는 확신이
  없으면 실행하지 않고 되묻게 설계됐습니다:

  > 🗣 "지우네 소풍 사진 보내줘"
  > 🧠 (개별 가정인지 반 전체인지 확신 부족) → **"지우 가정에만 보낼까요, 반 전체 학부모께
  > 보낼까요?"** — 실수 대신 확인.

- **⭐ 내 기여가 자산** — 오늘 가르친 "우리끼린 가정통신문이라고 해요" 하나가, 내일
  **모든 선생님**의 킨더버스를 그 말로 알아듣게 만듭니다. 한 번 가르친 것은 사라지지 않아요.

### 그냥 챗봇(범용 AI)을 붙이면 안 되나요?

현장의 말은 짧고 애매합니다. **"이거 좀 정리해줘"** — 사진첩을 보며 말하면 **사진 정리**,
자료함에서면 **자료함 정리**, 글을 쓰다 말하면 **문장 다듬기**입니다. 말만 듣는 범용 챗봇은
이 구별을 못 합니다. 킨더브레인은 **말 + 화면 맥락 + 직전 행동 + 선생님 말버릇**을 묶어
속뜻을 알아내도록 훈련되고, 그 실력을 시험으로 확인받습니다.

| | 범용 챗봇을 붙인 서비스 | 킨더브레인 |
|---|---|---|
| 애매한 말 | "이거 좀"이 뭔지 모름 → 되물어도 엉뚱하게 | 화면·직전 행동·말버릇으로 구별 |
| 정확도 | **증명 못 함** ("잘 되는 것 같아요") | **주 1회 전문가 시험**으로 채점 (목표 80%) |
| 위험한 순간 | 자신만만하게 실행 | 위험 7가지는 **오발률 2% 이하** 기준 통과 필수, 확신 없으면 되물음 |
| 개인화 | 누구에게나 똑같이 답함 | 쓸수록 내 말버릇에 맞춰짐 (상한선 있음) |

### 이게 왜 서비스 경쟁력이 되나요?

1. **현장 데이터 해자** — 실제 선생님들이 가르친 발화·의도 데이터는 밖에서 돈 주고도 못 삽니다.
   쓰는 교사가 늘수록 데이터가 쌓이고 → 서비스가 편해지고 → 더 많이 쓰게 되는 **선순환**은
   경쟁사가 복제할 수 없습니다. (범용 모델은 누구나 살 수 있지만, 이 데이터는 우리 현장에서만
   나옵니다.)
2. **증명하는 정확도** — "저희 AI 똑똑해요"가 아니라 **"전문가 출제 시험에서 기준을 넘긴
   버전만 내보냅니다"**라고 말할 수 있습니다. 성적이 나빠진 새 버전은 자동으로 승격이
   거부됩니다. 원장님·학부모·기관을 설득하는 언어 자체가 달라집니다.
3. **안전을 숫자로** — 되돌릴 수 없는 행동(학부모 전송·출결·삭제 등 7가지)은 별도 안전
   시험(오발률 기준)을 통과해야만 서비스에 연결됩니다. 유아 데이터를 다루는 서비스에서
   이 신뢰가 곧 선택의 이유입니다.

### 그래서, 선생님의 5분

뇌는 저절로 똑똑해지지 않습니다. 컴퓨터가 만든 연습문제는 뼈대일 뿐 — **진짜 교과서는
선생님의 답뿐입니다.**

- **강화하기 한 판 = 문제 6개 = 교과서 6줄** — 뇌가 즉시 커지고(공부량), 다음 시험에서
  실력으로 증명됩니다.
- **시험 문항 검수(O/X) 몇 분** — 뇌의 실력을 재는 자(시험지)가 생깁니다. 시험지가 있어야
  뇌가 밝아질 수 있어요.

오늘의 5분이, 내일 모든 교사의 몇 시간이 되어 돌아옵니다.

## 1. 한 문장으로 말하면

> **선생님의 말을 알아듣는 인공지능 "뇌"를 키우고, 그 뇌가 얼마나 자랐는지 눈으로 보여주는 화면입니다.**

나중에 선생님이 프로그램에게 이렇게 말하면 —

> 🗣️ "어제 민준이 관찰한 거 정리해서 알림장에 넣어줘"

프로그램이 스스로 알아들어야 합니다.

> 🧠 "아! ① 관찰 기록을 찾고 → ② 문서로 정리하고 → ③ 학부모께 보내려는 거구나"

이렇게 **말의 속뜻(의도)을 알아차리는 능력**을 키우는 것이 이 서비스의 목표입니다.

### 🌱 지금은 자라는 중이에요

이 뇌는 완성품이 아니라 **배우는 중인 아이**와 같습니다.

- 처음: 어둡고 조용함 (잠자는 뇌) ← **지금 화면이 어두운 이유!**
- 배우면: 점점 알록달록 풍성해짐
- 시험을 보면: 성적만큼 반짝반짝 빛남

**화면이 어두운 것 = 고장 ❌ / 아직 배우는 중 ⭕**

## 2. 뇌의 7개 영역 — 색깔로 나뉜 업무 지도

뇌를 돌려보면 7가지 색 영역이 보입니다. 유치원 업무를 7가지로 나눈 것입니다.

| 색 | 영역 | 이런 일을 배워요 |
|---|---|---|
| 🟢 초록 | **놀이** | "이 놀이 더 재밌게 넓혀줘", "다음 활동으로 넘어가자" |
| 🔵 하늘 | **관찰** | "민준이 오늘 어땠는지 요약해줘", "이 행동 지켜봐줘" |
| 🟠 주황 | **문서** | "알림장 초안 써줘", "맞춤법 봐줘" |
| 🟣 보라 | **시각** | "사진 정리해줘", "이 사진 예쁘게 잘라줘" |
| 🩷 분홍 | **소통** | "학부모께 안내 보내줘", "회신 모아줘" |
| 🟡 노랑 | **운영** | "자료 찾아줘", "일정 잡아줘", "자료함 정리해줘" |
| 🩵 청록 | **성찰** | "오늘 하루 돌아보자", "도움이 필요한 아이 있을까?" |

- 영역 안의 **동그란 점 하나 = 뇌가 배우는 "의도" 하나** (예: 노랑 영역의 한 점 = "자료 검색")
- 지금은 영역마다 9개씩, **모두 63개**의 의도를 배우고 있어요

## 3. 뇌는 어떻게 배우나요? — 데이터가 모이는 4가지 길

뇌를 가르치려면 **"이런 말 = 이런 뜻"** 예시가 아주 많이 필요합니다. 예시는 4가지 길로 모입니다.

### 길 ① 컴퓨터가 만드는 연습문제 (자동)

컴퓨터가 유치원에서 나올 법한 말을 **아주 많이** 지어냅니다.

> 만든 예: "블록 놀이 사진 좀 찍어서 정리해줄래?" → 정답: 사진 정리

- 👍 빠르게 많이 만들 수 있어요
- 👎 진짜 선생님 말과 다를 수 있어서 **"컴퓨터 생성" 등급**으로 낮게 취급해요
- ✅ 여러 인공지능이 서로 검토하고, 일부러 트집 잡는 검토까지 통과해야 저장돼요

### 길 ② 선생님이 직접 가르치기 ⭐ 제일 귀한 데이터!

화면에서 약한 점을 골라 **[🚀 강화하기]** 를 누르면 짧은 문제 풀이 게임이 열립니다.

**선생님이 고른 답 하나하나가 그대로 뇌의 교과서가 됩니다.**
컴퓨터가 만든 예시보다 훨씬 귀하게 취급돼요.

### 길 ③ 두 사람이 검증한 GOLD (금색 데이터)

어떤 예시가 정말 믿을 만한지 **두 명 이상이 따로따로 검토해서 똑같이 판단**해야만 **GOLD(금) 등급**이 됩니다.

- 한 명이 "확실해요!"라고 해도 ❌ — 반드시 2인 이상 일치 ⭕
- 화면에서 반짝이는 **금색 입자**가 바로 이 GOLD 데이터예요

### 길 ④ 전문가가 내는 시험 문제

유아교육 전문가가 **뇌 실력을 재는 시험지**를 만듭니다.

> 🔒 중요한 규칙: **시험 문제로는 절대 공부시키지 않아요.**
> 시험지를 미리 보고 외우면 실력이 아니니까요. 컴퓨터가 만든 문제도 시험지에 못 들어갑니다. (시스템이 자동으로 막아요)

## 4. 이 화면의 약속 — "모르면 모른다고 말해요"

이 서비스의 가장 중요한 철학입니다. 화면은 절대 거짓말을 하지 않아요.

| 약속 | 뜻 |
|---|---|
| **빛남 = 시험 성적만** | 공부를 아무리 많이 해도 점이 **커질 뿐**, 시험을 봐야만 빛나요 |
| **모르면 "—"** | 아직 안 잰 점수는 0%가 아니라 "—"로 써요 |
| **가짜면 "MOCK"** | 데모 화면에는 반드시 MOCK(가짜) 딱지가 붙어요 |

> 📚 비유: 문제집 100권을 푼 아이라도, 시험을 본 적이 없다면
> "실력 검증 전"이라고 말하는 게 정직하지요. 이 화면이 그렇게 동작해요.

## 5. 뇌의 성장 6단계

바닥의 **동심원 불빛**이 지금 단계를 알려줍니다. (켜진 원 개수 = 도달한 단계)

| 단계 | 이름 | 쉽게 말하면 |
|---|---|---|
| 0 | 잠듦 (Dormant) | 아직 시험 전 — 공부하는 중 💤 |
| 1 | 불꽃 (Spark) | 첫 시험을 봤어요 ✨ |
| 2 | 군집 깨어남 | 한 영역의 절반 이상이 시험을 봤어요 |
| 3 | 영역 가동 | 영역 평균 성적이 70%를 넘었어요 |
| 4 | 영역 간 흐름 | 영역끼리 헷갈리던 것이 좋아지는 중 |
| 5 | **뇌 전체 공명** | 전체 성적 80% 달성 — 최종 목표! 🎉 |

바닥 바깥의 **호(둥근 게이지)** 는 전체 시험 점수예요. 시험 전에는 아예 없습니다.

---

## 6. 선생님의 정보는 안전한가요?

- 훈련에서 고른 답은 **뇌 교육에만** 쓰여요
- 이름 대신 **암호화된 표식**으로 저장돼요
- "선생님 성향" 분석은 개인이 아니라 **여러 명 묶음 단위**로만 해요 (인원이 적으면 아예 계산 안 함)
- 성향이 뇌 판단에 주는 영향은 **상한선**이 있어요 — 뇌가 특정 성향에 치우치지 않아요

## 7. 한 장 요약

```
 🧠 선생님 말을 알아듣는 뇌를 키웁니다

 배우는 길 4가지                     화면에 보이는 것
 ──────────────                    ──────────────
 ① 컴퓨터 연습문제 (많이)      →     점 크기 = 공부한 양
 ② 선생님 가르침 ⭐ (귀하게)   →     입자 구름 = 공부 흔적
 ③ 2인 검증 GOLD (금색)      →     금색 반짝임 = 검증 데이터
 ④ 전문가 시험지 (실력 측정)   →     점의 빛남 = 시험 성적 ← 오직 이것만!

 약속: 모르면 "—" · 가짜면 "MOCK" · 어두우면 "공부 중"
```
