## 1. 화면은 4부분이에요

```
┌──────────┬────────────────┬──────────┐
│  왼쪽     │                │  오른쪽    │
│          │                │          │
│ 전체 점수  │    3D 뇌 🧠    │ 도구 카드  │
│ 영역 목록  │  (돌려보세요!)   │ 점 상세    │
│          │                │          │
├──────────┴────────────────┴──────────┤
│        [2D 지도로 보기]  ← 하단 중앙     │
└───────────────────────────────────────┘
```

| 위치 | 무엇이 있나요 |
|---|---|
| **왼쪽** | 뇌 전체 점수, 7개 영역 목록 (누르면 영역 상세) |
| **가운데** | 3D 뇌 — 마우스로 돌리고, 만지는 공간 |
| **오른쪽** | 도움말 · 성향 오버레이 · 혼동 관계 · 인코딩 범례 카드 + (점을 고르면) 점 상세 |
| **하단 중앙** | 3D ↔ 2D 지도 전환 버튼 |

## 2. 뇌 읽는 법 — 이 표 하나면 끝!

화면의 모든 것에는 뜻이 있어요. (오른쪽 "인코딩 범례" 카드 → **펼치기**를 눌러도 볼 수 있어요)

| 눈에 보이는 것 | 뜻 |
|---|---|
| 점이 **크다** | 공부를 많이 했어요 |
| 점이 **빛난다** | 시험 성적이 좋아요 — **시험 전엔 모두 어두워요(정상!)** |
| 점 둘레 **흰 고리** | 새로 공부했어요 — 재시험 기다리는 중 |
| **입자 구름이 알록달록** | 그 영역이 공부한 만큼 선명하고 풍성해져요 |
| **금색 반짝임** | 2인 검증 GOLD·전문가 데이터 |
| **점선 연결** | "헷갈릴 수 있다"는 예상 (아직 측정 전) |
| **실선 + 깜빡임** | 시험에서 진짜 헷갈린 것 (빠를수록 자주 헷갈림) |
| **바닥 동심원** | 성장 단계 (켜진 개수 = 단계) |
| **바닥 바깥 호** | 전체 시험 점수 (시험 전엔 없음) |

> 💡 **뇌가 어두워요?** 고장이 아니에요! 시험(Arena) 전에는 모든 점이 어둡습니다.
> 범례 카드에도 "뇌가 어두운 것이 정상이에요"라고 적혀 있어요.

## 3. 기본 조작 — 30초면 배워요

| 하고 싶은 것 | 이렇게 하세요 |
|---|---|
| 뇌 돌리기 | 마우스 **드래그** (그냥 두면 천천히 저절로 돌아요) |
| 크게/작게 | 마우스 **휠** |
| 영역 구경 | 영역에 **마우스 올리기** → 그 영역만 반짝 살아나요 |
| 영역 정보 | 영역 **이름표 클릭** (왼쪽 목록에서 눌러도 돼요) |
| 점 자세히 | **점 클릭** → 오른쪽에 상세 정보 |
| 선택 취소 | 빈 곳 클릭 |
| 가볍게 보기 | 하단 **[2D 지도로 보기]** |
| 도움말 | 오른쪽 위 **[❔ 도움말]** |

## 4. 점을 클릭하면 — 오른쪽 "건강 검진표" 읽기

### ① 기본 수치 (KEY METRICS)

| 항목 | 뜻 |
|---|---|
| Evidence Total | 공부한 데이터 개수 |
| Gold | 그중 GOLD(2인 검증) 개수 |
| Diversity | 공부 재료가 얼마나 다양한지 |
| Exemplars | 뇌가 참고하는 대표 예문 수 |

### ② 대표 혼동 관계 — "누구랑 헷갈리나"

- **가설**: "헷갈릴 수 있겠다"고 예상된 것 (아직 측정 전)
- **확정 + %**: 시험에서 실제로 헷갈린 비율

3D에서는 선택한 점에서 **점선 부채꼴**이 뻗고, 선을 따라 **빛이 흐르고**, 연결된 점 위에 **"왜 연결됐는지" 카드**가 떠요.

### ③ 왜 약한가 (WHY THIS NODE IS WEAK)

| 항목 | 쉽게 말하면 |
|---|---|
| Ambiguous Language | 말 자체가 여러 뜻으로 읽히나? |
| Screen Context Coverage | 여러 화면 상황을 연습해 봤나? |
| Persona Diversity | 여러 성향의 선생님께 배웠나? |
| Gold Data | 검증된 데이터가 충분한가? |

**HIGH = 위험/부족 · LOW = 양호 · "—" = 아직 잴 수 없음**

## 5. 뇌 가르치기 — 강화하기 ⭐ 제일 중요해요!

선생님이 뇌를 직접 가르치는 기능입니다. 5분이면 충분해요.

### 따라 하기

**1️⃣ 약한 점 찾기**
왼쪽 영역 상세의 **TOP WEAK NODES** 목록을 보거나, 뇌에서 작고 어두운 점을 클릭

**2️⃣ [🚀 강화하기] 클릭**
잠시 후 **훈련 브리핑**이 떠요 — 몇 문제인지, 무엇을 겨냥하는지

**3️⃣ 훈련 방식 고르기**

| 방식 | 하는 일 |
|---|---|
| 의도 알아맞히기 | 문장 보고 "무슨 뜻일까?" 고르기 |
| 알맞은 의미 고르기 | 헷갈리는 두 뜻 중 맞는 쪽 고르기 |
| 바로잡기 연습 | 뇌가 틀린 걸 바로잡아 주기 |

> "데이터 공장에 시나리오 생성만 요청했어요"라고 뜨면?
> → 이번 건은 자동으로 처리돼서 선생님이 풀 문제가 없다는 뜻이에요.

**4️⃣ 문제 풀고 [제출하기]**

**5️⃣ 제출하면 이렇게 됩니다** (자동)

```
내 답 제출
  → 뇌의 교과서로 저장 ("사람이 확인함" 등급 ⭐)
  → 점이 즉시 커지고 입자가 늘어남 (공부량 반영)
  → 점 둘레 흰 고리 켜짐 ("재시험 기다려요")
  → 다음 시험 후 → 성적이 오르면 그때 빛나요 ✨
```

> ⚠️ **제출해도 점이 바로 안 빛나요!** 정상이에요.
> 공부 → 점이 커짐 / 시험 → 점이 빛남. 순서가 달라요.

## 6. 내 답은 어디로 가나요?

선생님이 고른 답 하나의 여행:

```
내 답 1개
  ↓
"문장 + 상황 + 내 선택"으로 저장
  ↓
등급표에서 높은 자리를 받아요:
  컴퓨터 생성 < 행동 신호 < ⭐내 답(사람 확인) < GOLD(2인 검증) < 전문가
  ↓
점 크기·입자에 바로 반영
  ↓
다음 시험에서 실력 측정 → 그때 빛남
```

**알아두세요**
- 이름 대신 **암호화된 표식**으로 저장돼요
- 일부러 어려운 "함정 문제" 답은 **따로 보관**돼서 일반 통계를 흐리지 않아요
- GOLD가 되려면 **꼭 2인 이상**이 따로 검토해서 일치해야 해요
- 시험지에는 컴퓨터 생성 문제가 **못 들어가요** (시스템이 차단)

## 7. 자주 묻는 질문 (FAQ)

**Q. 뇌가 온통 어두워요. 고장인가요?**
→ 정상이에요! 빛 = 시험 성적뿐인데, 아직 시험 전이거든요. "Dormant(잠듦)" 표시가 그 뜻이에요.

**Q. 점수가 "—%"로 나와요.**
→ "아직 안 재봤다"는 뜻이에요. 0%가 아니에요. 이 화면은 모르는 걸 지어내지 않아요.

**Q. 훈련을 했는데 점이 안 빛나요.**
→ 정상! 훈련 → 커짐 + 흰 고리 / 시험 → 빛남. 다음 시험 후에 반영돼요.

**Q. "백엔드 미연결"이 떠요.**
→ 데이터 서버가 꺼져 있어요. 관리자에게 알려주세요. [데모 노드 보기]를 누르면 구경은 할 수 있어요 (가짜 데이터라 MOCK 딱지가 붙어요).

**Q. 점선이 잔뜩 보여요. 뭐예요?**
→ "이 둘은 헷갈릴 수 있다"는 예상 목록이에요. 평소엔 선택한 점 것만 보이고, 혼동 관계 카드의 [전체 보기]로 다 볼 수 있어요.

**Q. "성향 오버레이"가 꺼져 있어요.**
→ 선생님 성향 데이터가 충분히 모이면 켜져요. 지금은 "미실행" 상태가 맞아요.

**Q. 제 이름이 저장되나요?**
→ 아니요. 암호화된 표식으로 저장되고, 성향 분석도 개인이 아닌 묶음 단위로만 해요.

**Q. 3D가 느리거나 안 보여요.**
→ 하단 [2D 지도로 보기]를 누르세요. 기능은 똑같아요. 3D가 안 되는 기기에선 자동으로 2D가 열려요.

## 8. 좋은 답을 만드는 요령 🌱

1. **평소 말투로** 생각하며 골라주세요 — 뇌는 진짜 선생님 말을 배워야 해요
2. **모르면 넘어가도 돼요** — 억지 답 10개보다 확실한 답 1개가 귀해요
3. **헷갈리는 문제일수록 신중하게** — 그 답이 뇌의 약점을 정확히 고쳐줘요
4. **짧게, 자주** — 한 번에 5분이면 충분해요

선생님의 답 하나하나가 이 뇌의 교과서가 됩니다. 고맙습니다! 💛
