# Kinder Intent Brain — 사용설명서 (선생님용)

> 이 문서는 **"어떻게 써요?"** 에 답합니다.
> 서비스가 무엇인지·왜 키우는지는 『서비스 설명서』(service-overview.md)를 봐주세요.

---

## 1. 🧭 처음이라면 — 순서는 이렇게

이 화면에서 제일 많이 하는 오해가 "가르쳤는데 왜 안 밝아져요?"입니다. 순서가 있어요.

| 순서 | 누가 | 무엇을 | 그러면 |
|---|---|---|---|
| **① 시험지 만들기** | 검수자 두 분 | 시험 문항 작성·검수 — 상단 **[📄 시험지 검수]** 또는 도움말 → 시험 문항 만들기. **위험 의도 7개부터** 권장 | 뇌 실력을 잴 "자"가 생김 |
| **② 채점 (주 1회)** | 운영자 | **[🎯 채점 실행]** 버튼(대시보드·검수창, 비밀번호 필요) | **뇌가 처음으로 밝아짐** (지금 실력만큼) |
| **③ 가르치기 (상시)** | 모든 선생님 | **강화하기 · 즉석 문답** | 다음 채점 때 더 밝아짐 |

> 💡 외울 건 한 줄: **공부 = 점이 커짐 · 시험 = 점이 빛남.**
> 시험지가 없으면 아무리 가르쳐도 빛나지 않아요(고장 아님) — 그래서 ①이 먼저입니다.

## 2. 화면 한눈에

```
┌───────────────────────────────────────────────┐
│ KINDER BRAIN   —% 점수   [📄 시험지 검수]      │ ← 상단 바
│                          [💬 즉석 문답] [❔]   │
├──────────┬──────────────────────┬─────────────┤
│  왼쪽     │                      │  오른쪽      │
│ 전체 점수  │      3D 뇌 🧠        │ 점 상세      │
│ 영역 목록  │   (돌려보세요!)       │ 진단·강화    │
├──────────┴──────────────────────┴─────────────┤
│          [2D 지도로 보기]  ← 하단 중앙          │
└───────────────────────────────────────────────┘
```

| 위치 | 무엇이 있나요 |
|---|---|
| **상단 바** | 가운데 전체 점수("—%"면 시험 전) · 오른쪽 **[📝 시험지 작성] [💬 즉석 문답] [❔ 도움말]** |
| **왼쪽** | 뇌 전체 점수, 8개 영역 목록 (누르면 영역 상세) |
| **가운데** | 3D 뇌 — 마우스로 돌리고, 만지는 공간 |
| **오른쪽** | (점을 고르면) 점 상세 · 왜 약한지 진단 · [🚀 강화하기] |
| **하단 중앙** | 🧠 3D 뇌 ↔ 📊 대시보드 전환 · 성향 렌즈 · 헷갈림 연결 |

## 3. 뇌 읽는 법 — 이 표 하나면 끝!

| 눈에 보이는 것 | 뜻 |
|---|---|
| 점이 **크다** | 공부를 많이 했어요 |
| 점이 **빛난다** | 시험 성적이 좋아요 — **시험 전엔 모두 어두워요(정상!)** |
| 점 둘레 **흰 고리** | 새로 공부했어요 — 재시험 기다리는 중 |
| **입자 구름** | 공부한 만큼 선명하고 풍성해져요 |
| **금색 반짝임** | 2인 검증 GOLD·전문가 데이터 |
| **점선 연결** | "헷갈릴 수 있다"는 예상 (아직 측정 전) |
| **실선 + 깜빡임** | 시험에서 진짜 헷갈린 것 (빠를수록 자주) |
| **바닥 동심원** | 성장 단계 (켜진 개수 = 단계) |
| **바닥 바깥 호** | 전체 시험 점수 (시험 전엔 없음) |

## 5. 💬 즉석 문답 — 아무 말이나 시켜보세요

뇌에게 **지금 바로** 한마디를 던져보고, 알아듣는지 확인하고, 그 자리에서 가르치는 기능입니다.

1. 상단 **[💬 즉석 문답]** → 평소 말 그대로 입력 — 예: "지우 오늘 병결로 해줘"
2. 뇌가 **후보 3~4개**로 추측해요
3. 판정하기:
   - 맞으면 → 그 후보 선택 (**확인** — 뇌에게 "그래 맞아")
   - 틀리면 → 정답 후보 선택 (**교정** — **제일 귀한 가르침**)
   - 후보에 없으면 → **전체 의도 검색**으로 정답 지정
   - 70개 어디에도 없는 진짜 새 일이면 → **"새 의도 제안"** (자동 등록되지 않고 확장
     후보로만 쌓여요)

> 💡 내 판정은 곧바로 교과서가 됩니다(점 커짐 + 흰 고리). 단 **점수(빛)는 안 변해요** —
> 그건 시험만 바꿔요.
>
> 📝 **"시험 문제로 출제" 토글**을 켜면 이 문답은 훈련에 쓰지 않고 **시험지 검수
> 대기열**로 갑니다. 같은 문장을 공부와 시험 양쪽에 쓸 수 없기 때문이에요(암기 측정 방지).

## 6. 🚀 강화하기 — 약한 점 집중 과외

**1️⃣ 약한 점 찾기** — 왼쪽 **TOP WEAK NODES** 목록 또는 작고 어두운 점 클릭
**2️⃣ [🚀 강화하기] 클릭** → 훈련 브리핑(몇 문제, 무엇을 겨냥) 확인
**3️⃣ 방식 고르기**

| 방식 | 하는 일 |
|---|---|
| 의도 알아맞히기 | 문장 보고 "무슨 뜻일까?" 고르기 |
| 알맞은 의미 고르기 | 헷갈리는 두 뜻 중 맞는 쪽 고르기 |
| 바로잡기 연습 | 뇌가 틀린 걸 바로잡아 주기 |

**4️⃣ 문제 풀고 [제출하기]**

```
제출하면 (자동):
내 답 → 교과서로 저장(⭐사람 확인 등급) → 점이 즉시 커지고 흰 고리 점등
      → 다음 시험 후 성적만큼 빛남 ✨
```

> ⚠️ **제출해도 바로 안 빛나요 — 정상!** 공부=커짐, 시험=빛남.
> "데이터 공장에 시나리오 생성만 요청했어요"가 뜨면 이번 건은 자동 처리라 풀 문제가
> 없다는 뜻이에요.

## 7. 📄 시험 문항 만들기·검수 — 두 가지 방법

뇌 실력을 재는 시험지를 채우는 일입니다. **되돌릴 수 없는 위험 의도 7개**(학부모 전송·출결·
삭제 등)부터 채우는 걸 권장해요 — 안전 게이트가 이 7개에 각 30문항을 요구합니다.

| 방법 | 이럴 때 | 흐름 |
|---|---|---|
| **웹에서 바로** | 자리에서 몇 문항씩 | **[📄 시험지 검수]** → ① 문항 작성 탭: 질문 + 내 점수(1~5) 제출 → ② 다른 사람이 "검수 대기" 탭에서 **점수 안 보고(blind)** 1~5 → 일치도 통과 시 [등록] |
| **엑셀로 한꺼번에** | 여럿이 나눠 대량으로 | 도움말 → **시험 문항 만들기**에서 양식 받기(⭐**CRITICAL 7개 먼저** 권장, 또는 70×10 전체) → 질문 쓰고 두 분이 O/X → 업로드 → 검증 → 등록 확정 |

**두 방법 모두 같은 규칙:**
- **서로 다른 2인**이 검수 (같은 사람 두 번 = 무효, 시스템이 막아요)
- **사람이 쓴 진짜 질문만** (AI로 만든 문장 금지 — 점수가 무의미해져요)
- **공부용 문장과 겹치면 자동 거부** (겹치면 "암기 측정"이 되니까)
- 일치도(kappa)는 **자동 계산** — 직접 셈할 필요 없어요

> 상세 작성 요령은 『시험 문항 만드는 법』(exam-authoring.md)과
> 『CRITICAL 7 출제 가이드』(exam-authoring-critical7.md)를 봐주세요.

## 8. 점을 클릭하면 — 오른쪽 "건강 검진표" 읽기

### ① 기본 수치 (KEY METRICS)

| 항목 | 뜻 |
|---|---|
| Evidence Total | 공부한 데이터 개수 |
| Gold | 그중 GOLD(2인 검증) 개수 |
| Diversity | 공부 재료가 얼마나 다양한지 |
| Exemplars | 뇌가 참고하는 대표 예문 수 |

### ② 대표 혼동 관계 — "누구랑 헷갈리나"

- **가설**: "헷갈릴 수 있겠다"고 예상된 것 (아직 측정 전)
- **확정 + %**: 시험에서 실제로 헷갈린 비율

### ③ 왜 약한가 (WHY THIS NODE IS WEAK)

| 항목 | 쉽게 말하면 |
|---|---|
| Ambiguous Language | 말 자체가 여러 뜻으로 읽히나? |
| Screen Context Coverage | 여러 화면 상황을 연습해 봤나? |
| Persona Diversity | 여러 성향의 선생님께 배웠나? |
| Gold Data | 검증된 데이터가 충분한가? |

**HIGH = 위험/부족 · LOW = 양호 · "—" = 아직 잴 수 없음**

## 9. 내 답은 어디로 가나요?

```
내 답 1개
  ↓ "문장 + 상황 + 내 선택"으로 저장
  ↓ 등급: 컴퓨터 생성 < 행동 신호 < ⭐내 답(사람 확인) < GOLD(2인 검증) < 전문가
  ↓ 점 크기·입자에 바로 반영
  ↓ 다음 시험에서 실력 측정 → 그때 빛남
```

- 이름 대신 **암호화된 표식**으로 저장돼요
- 일부러 어려운 "함정 문제" 답은 **따로 보관**돼 일반 통계를 흐리지 않아요
- 시험지에는 컴퓨터 생성 문제가 **못 들어가요** (시스템이 차단)

## 10. 자주 묻는 질문 (FAQ)

**Q. 뭐부터 해야 해요?**
→ 시험지부터! §1 "처음이라면" 순서(①시험지 → ②채점 → ③가르치기)대로.

**Q. 채점은 누가 어떻게 돌리나요?**
→ **운영자**가 돌려요. 두 가지 방법:
- **웹(권장)**: 📊 대시보드의 PERFORMANCE 옆 또는 📄 시험지 검수창 상단의 **[🎯 채점 실행]**
  버튼 → **운영자 비밀번호** 입력 → 몇 분 뒤 점수·밝기가 자동 반영. (비밀번호는 운영자에게,
  서버 환경변수 `ARENA_PIN`으로 변경 가능)
- **명령어**: `python scripts/run_arena.py --commit` (같은 동작)

주기는 **주 1회** 권장이고, 시험 문항을 새로 채웠을 때도 한 번 돌리면 좋아요. 참고: 서버의
AI가 실 모델이 아니면(mock) 가짜 점수 방지를 위해 채점이 거부돼요.

**Q. 뇌가 온통 어두워요. 고장인가요?**
→ 정상! 빛 = 시험 성적뿐인데 아직 시험 전이에요. "Dormant(잠듦)"가 그 뜻이에요.

**Q. 점수가 "—%"로 나와요.**
→ "아직 안 재봤다"는 뜻. 0%가 아니에요. 이 화면은 모르는 걸 지어내지 않아요.

**Q. 훈련(강화하기)을 했는데 점이 안 빛나요.**
→ 정상! 훈련 → 커짐 + 흰 고리 / 시험 → 빛남. 다음 시험 후에 반영돼요.

**Q. 즉석 문답에서 답하면 점수에 반영되나요?**
→ 공부량(크기·고리)엔 즉시 반영, 점수(빛)는 다음 시험에서만 바뀌어요.

**Q. "백엔드 미연결"이 떠요.**
→ 데이터 서버가 꺼져 있어요. 관리자에게 알려주세요. [데모 노드 보기]는 구경용(MOCK 딱지).

**Q. 점선이 잔뜩 보여요. 뭐예요?**
→ "이 둘은 헷갈릴 수 있다"는 예상이에요. 평소엔 선택한 점 것만 보이고, [전체 보기]로 다 볼 수 있어요.

**Q. "성향 오버레이"가 꺼져 있어요.**
→ 선생님 성향 데이터가 충분히 모이면 켜져요. 지금은 "미실행"이 맞아요.

**Q. 제 이름이 저장되나요?**
→ 아니요. 암호화된 표식으로 저장되고, 성향 분석도 개인이 아닌 묶음 단위로만 해요.

**Q. 3D가 느리거나 안 보여요.**
→ 하단 [2D 지도로 보기]를 누르세요. 기능은 똑같아요.

## 11. 좋은 답을 만드는 요령 🌱

1. **평소 말투로** — 뇌는 진짜 선생님 말을 배워야 해요
2. **모르면 넘어가기** — 억지 답 10개보다 확실한 답 1개
3. **헷갈릴수록 신중히** — 그 답이 뇌의 약점을 정확히 고쳐요
4. **짧게, 자주** — 한 번에 5분이면 충분해요

선생님의 답 하나하나가 이 뇌의 교과서가 됩니다. 고맙습니다! 💛
